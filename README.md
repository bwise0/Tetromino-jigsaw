# Tetromino Jigsaw Puzzle (테트로미노 직소 퍼즐)

테트로미노(Tetromino) 조각들을 이용해 숨겨진 이미지를 완성하는 독특한 직소 퍼즐 게임입니다.

## 🎮 게임 개요
이 게임은 고전적인 테트리스 조각 모양을 활용한 퍼즐 게임입니다. 단순히 모양을 맞추는 것을 넘어, 각 조각에 분할된 이미지가 포함되어 있어 조각들을 올바른 위치와 방향으로 맞추면 하나의 아름다운 그림이 완성됩니다.

## 🕹️ 게임 방법
1. **조각 이동**: 마우스나 터치로 조각을 드래그하여 원하는 위치로 옮길 수 있습니다.
2. **조각 회전**: 조각을 가볍게 클릭(터치)하면 시계 방향으로 90도씩 회전합니다.
3. **조각 병합 (Snap)**: 두 조각이 서로 정답 위치 관계에 있고 회전 각도가 일치하면, 자석처럼 서로 달라붙어 하나의 그룹이 됩니다.
4. **목표**: 화면 중앙의 점선으로 표시된 영역 안에 모든 조각을 올바른 방향으로 배치하여 이미지를 완성하세요.
5. **초기화**: 상단의 'Reset' 버튼을 누르면 새로운 이미지와 함께 조각들이 재배치됩니다.

## ✨ 주요 기능
- **이미지 분할 렌더링**: 하나의 이미지를 테트로미노 조각의 형태에 맞춰 동적으로 분할하여 보여줍니다.
- **지능형 병합 시스템**: 조각 간의 상대적 위치와 회전값을 계산하여 정답일 경우 자동으로 그룹화합니다.
- **부드러운 애니메이션**: Framer Motion을 사용하여 조각의 이동, 회전, 병합 과정을 부드럽게 연출합니다.
- **반응형 디자인**: 다양한 화면 크기에서도 쾌적하게 플레이할 수 있도록 최적화되었습니다.

## 🛠️ 개발 내역 (Update History)
최근 업데이트를 통해 다음과 같은 버그 수정 및 성능 개선이 이루어졌습니다:

1. **회전 중심축 및 경계 보정**
   - 조각 회전 시 중심축이 어긋나던 문제를 해결하여 제자리에서 자연스럽게 회전하도록 수정했습니다.
   - 조각이 화면 밖으로 튀어나가지 않도록 경계 충돌 방지 로직을 추가했습니다.

2. **드래그-회전 간섭 수정**
   - 드래그 이동 후 손을 뗄 때 의도치 않게 조각이 회전하던 현상을 `onTap` 이벤트를 도입하여 해결했습니다.

3. **병합 시 위치 안정화**
   - 조각들이 합쳐질 때 위치가 갑자기 튀거나 흔들리는(Jumping) 현상을 수정했습니다.
   - 그룹 내 기준 조각(Reference Block)을 고정하고 회전 피벗을 안정화하여 부드러운 병합 경험을 제공합니다.

## 💻 기술 스택
- **Frontend**: React, TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Assets**: Lorem Picsum (Random Image API)
